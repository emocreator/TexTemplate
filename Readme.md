# LaTeX Document Template

My structured LaTeX template

## Usage

1. Edit `Root.tex` for document metadata and structure
2. Add chapters in `Chapters` directory
3. Manage references in `ref.bib`

## Compilation

```bash
pdflatex Root.tex
bibtex Root
pdflatex Root.tex
```

## Dependencies

- TeX Live or MiKTeX
